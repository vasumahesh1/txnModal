txnModal - Examples

Basic Modal
Modal with nothing Overriden

Normal Modal
Basic Modal with Overlapped Close Button.sample-button
$('#normal-modal').txnModal({ 
	modalCloseHandlers : ['#closeModalBtn'] 
});
Normal CSS Overriden Modal
Basic Modal with Overriden CSS and Disables scroll to illustrate.
$('#css-modal').txnModal({ 
	modalCss : { 
		'height' : '450px',
		'width' : '750px',
		'left' : '50%',
		'top' : '50%',
		'margin-left' : '-375px',
		'margin-top' : '-225px',
		'border-radius' : '3px',
		'background' : '#f6f6f6'
	}
});
Auto Centering Modal
Auto Centers the Modal Depending on the CSS Property
// Both Centering
$('#auto-modal-1').txnModal({ 
	modalCss : { 
		'height' : '450px',
		'width' : '750px'
	},
	modalAutoCenter: true
});

// Only X Axis Centering
$('#auto-modal-2').txnModal({ 
	modalCss : { 
		'height' : '450px',
		'width' : '750px',
		'top' : '10px'
	},
	modalAutoCenter: true
});

// Only Y Axis Centering
$('#auto-modal-3').txnModal({ 
	modalCss : { 
		'height' : '450px',
		'width' : '750px',
		'left' : '10px'
	},
	modalAutoCenter: true
});
